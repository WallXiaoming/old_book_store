# old_book_store

校园二手图书交易

有以下功能：

- 二手图书的发布
- 二手图书的更新
- 二手图书的删除
- 评论
- 图书搜索
- 账号的注册
- 账号登录
- 密码找回
- 用户名与邮箱的修改
- 后台管理
- 评论审核


使用方式：

1. 克隆仓库
```
git clone https://github.com/WallXiaoming/old_book_store.git
```

2. 安装环境
```
pip install -r requirement.txt
```

3. 运行
```
python manage.py runserver
```
